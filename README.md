# OSDI Specification

**Open Services Delivery Interface (OSDI)**

Status: Early Working Draft (0.1.0-draft)

OSDI is an open, implementation-neutral specification describing how authorised service organisations interact securely with compatible smart places.

NHS365 is intended to be a reference implementation of this specification rather than the specification itself.
